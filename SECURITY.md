# Security Policy

## Supported Versions

The `main` branch is the supported development line.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

1. Open a private advisory report: `https://github.com/signalrust/modelsentry/security/advisories/new`
2. Include: affected version/commit, impact, reproduction steps, and logs or PoC when available
3. You will receive an acknowledgment within 72 hours
4. Confirmed issues are remediated with highest priority and disclosed after fix availability

## Disclosure Policy

- Coordinated disclosure is preferred
- We request 90 days before public disclosure for critical findings
- Credit will be provided to reporters unless anonymity is requested
